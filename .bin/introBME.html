TCP Killer

Barracuda is a utility to kill all TCP connections on a network. It works by intercepting network traffic, and forging RST packets of existing
connections to cause hangups. Works for both IPv4 connections and IPv6. Barracuda ( TCPkiller ) is a sneaky way to kill network connections.

Any targeted system will appear to work when examined through typical diagnostics - DNS will resolve, ICMP's will go through, and the network
card will be able to connect ( slow ) to the network-but no TCP connections will be sustained.
